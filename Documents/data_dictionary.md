# Data Dictionary

## Customer Support Dataset

| Column | Description |
|----------|------------|
| subject | Ticket subject |
| body | Customer message |
| answer | Support response |
| type | Ticket category |
| queue | Support queue |
| priority | Priority level |
| language | Ticket language |
| tag_1 to tag_8 | Issue tags |
