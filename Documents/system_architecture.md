# System Architecture

## Overview

The AI Business Value Assessment Platform consists of six major layers.

---

# Layer 1: Data Sources

## Customer Support Dataset

Provides:

* Ticket Information
* Support Categories
* Priorities
* Customer Messages

---

## Online Retail Dataset

Provides:

* Transactions
* Product Information
* Customer Purchases
* Revenue Data

---

## Amazon Reviews Dataset

Provides:

* Customer Reviews
* Ratings
* Product Feedback

---

# Layer 2: Data Processing Layer

Functions:

* Data Cleaning
* Missing Value Handling
* Feature Engineering
* Data Transformation

Outputs:

* Cleaned Analytical Datasets

---

# Layer 3: Analytics Layer

Modules:

## Customer Support Analytics

* Ticket Volume Analysis
* Queue Analysis
* Priority Analysis
* Language Analysis

## Customer Voice Intelligence

* Complaint Categorization
* Topic Modeling
* Sentiment Analysis

## Retail Analytics

* Revenue Analysis
* Product Analysis
* Customer Analysis
* Country Analysis

## Customer Segmentation

* RFM Analysis
* Customer Lifetime Value Analysis

---

# Layer 4: AI Business Value Layer

Modules:

* Opportunity Identification
* Automation Potential Assessment
* Initiative Scoring

Outputs:

* AI Opportunity Scores
* Initiative Assessments

---

# Layer 5: Decision Support Layer

Modules:

* MCDA Framework
* Initiative Ranking
* Recommendation Engine

Outputs:

* Investment Recommendations
* Priority Rankings

---

# Layer 6: Presentation Layer

## Power BI Dashboards

* Executive Dashboard
* Customer Voice Dashboard
* Revenue Dashboard
* AI Opportunity Dashboard

## Streamlit Application

* Interactive Assessment Platform
* Recommendation Generation
* Initiative Comparison
